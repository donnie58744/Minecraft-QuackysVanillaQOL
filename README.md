# Quackys Vanilla QOL

## Purpose

If you want Minecraft to be more immersive, less of a pain and running smoother; than this modpack is for you!

- The only block added to the game are the Chunk Loaders.
- Adds over 25 QOL mods to make Minecraft just that much better to play
- Adds basic Performance mods ([Sodium](https://modrinth.com/mod/sodium), [Lithium](https://modrinth.com/mod/lithium), etc...)
- Also includes BSL shaders as well as Primes HD Textures for better immersion
- More realistic audio experience
